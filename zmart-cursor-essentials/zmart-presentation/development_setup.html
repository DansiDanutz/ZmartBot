Development Environment Setup

Step-by-step guide for setting up the development environment for the Zmart Trading Bot Platform.

Prerequisites

Install: Docker & Docker Compose, Node.js (v18+), Python 3.11+, Git

Clone Repository

git clone https://github.com/your-org/zmart-platform.git
cd zmart-platform

Configuration

cp .env.example .env
nano .env # Edit settings

Start Infrastructure

docker-compose up -d postgres redis influxdb rabbitmq

Backend Setup

cd backend/zmart-api
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python src/main.py

Frontend Setup

cd frontend/zmart-dashboard
pnpm install && pnpm run dev

Verification

• Frontend: localhost:3000
• Backend API: localhost:5000
• PostgreSQL: localhost:5432
• Redis: localhost:6379
• InfluxDB: localhost:8086
• RabbitMQ: localhost:15672
